Privacy

Welcome to our support page for the DigiPuncher Face Recognition App. Clocking-in or out using your mobile device with the feature of face recognition is as easy as downloading the application to your phone, entering the registration code received in a text message and clock-in. The registration code configures the application to your employee number and company ID. If you wish to know more on how to use the Application watch our user guide below. If you need further assistance we are always available at 787-868-9100 or scs.digipuncher@gmail.com.

Application Store

The application can be downloaded depending on the operating system of your cell phone. If you have an Android phone you can download the application through Play Store, and if it's an Apple you can download it from the App Store.

The application contains a total storage capacity of 38 mb, make sure that your mobile phone has enough space to download it.

Registration Key

When you start the application, a screen will appear to place a registration key. This key will be given by Santos Computer & Services and will give you access to all the functionalities of the application.

Notes:
If for any reason you delete the application. You will lose the registration key and you need to contact Santos Computer & Services to have the key reactivated.

Main Menu

Work Codes
Work codes is a concept for companies in which the employee performs different types of tasks. In this case the work code is selected before the punch-out.
Latest Punches
In Latest Punches you can see all the punches made in the device up to two days ago. For more details about the punches you can visit our Iclock server page.
Add Employee
Add Employee allows your company's administrators to log in and get a list of all employees. The administrator will select the employee he wants to recruit for the face recognition system.
Unsent Punches
Unsent punches will appear all the punches that for some reason could not be sent at the time. Usually this happens due to internet connection problems.
Make Punch
Make Punch will open your mobile phone camera to take a picture of the employee or a group of employees.
