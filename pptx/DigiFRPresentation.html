DigiPuncher Face
recognition APP
A new way to generate a punch by taking a picture to one or multiple
employees at the same time using face recognition.
Registration key
• The first page when starting the application will be
to place the registration key, this key will be
generated in Digipuncher by an administrator or
human resources staff (more information in slide 3).
• The registration key will be unique for each device,
when it is deactivated it will not allow the device to
have the functionalities of the application.
https://www.digipuncher.com/{company}/pag_rechum/facerecognitiondevices.aspx
• To generate the registration key you
must go to the following web address
that is in the top title. You must replace
the area {company} with the name of
your company registered in Digipuncher.
• Name the device and make sure that
the checkbox to confirm the device is
clicked. Then press the green button.
Main menu
There are 5 buttons on the main menu
• The ‘Work codes’ button will load the company's work codes, select the
appropriate work code from the pop-up window. Please remember that
work codes are only used when clocking out.
• The ‘Latest punches’ button is to show the last punches confirmed from 2
days ago.
• The ‘Add employee’ button has a form to validate the user who is a
supervisor or manager, if the validation is correct, will return the
company employees to be recruited to the face recognition.
• The ‘Unsent punches’ button show you the list of punches that due to a
connection error could not be sent previously.
• Finally, the green button at the end of the screen is to make the punch,
press it and then press open camera to take the photo of the group.
